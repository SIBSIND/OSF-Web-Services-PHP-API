stuctWSF PHP API  1.00
The structWSF PHP API is a framework available to PHP developers to help them generating queries to any structWSF web service endpoint. Each structWSF web service endpoint has its own WebServiceQuery class in the structWSF PHP API. This class is used to generate any query, to send it to be endpoint of a structWSF instance and to get back a resultset. The resultset can then be manipulated by using the Resultset API. This same API can be used to transform the resultset in different formats.

setResultset (   $resultset)

Import an array that describes the resultset to use with this Resultset object.

The input resultset array has to be of format:

Array("record-uri" => Array( "type" => Array(URIs...), "prefLabel" => "preferred label", "altLabel" => Array(alternative label literals...), "prefURL" => "http://preferred-url.com", "description" => "some description of the record",

"other-data-attribute-uri" => Array( Array( "value" => "some value", "lang" => "language string of the value", "type" => "type of the value" ), Array( ... ) ), "more-data-attribute-uri": ...,

"other-object-attribute-uri" => Array( Array( "uri" => "some uri", "type" => "optional type of the referenced URI", "reify" => Array( "reification-attribute-uri" => Array("value of the reification statement"), "more-reification-attribute-uri" => ... ) ), Array( ... ) ) "more-object-attribute-uri": ... ),

"more-record-uri": ... )

Parameters:
mixed$resultsetResultset array to use in this Resultset object
